This is a responsive landing page built using React JS.

## 🚀 Features
- Responsive design
- Modern UI
- React Components
- CSS Styling
- Fast loading page

## 🛠 Technologies Used
- React JS
- JavaScript
- HTML5
- CSS3

## 📂 Project Structure
src/
  components/
  pages/
  App.js
  index.js

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/ATHILAKSHMI-JEEVA/landing-page.git

2. Install dependencies

npm install

3. Start the project

npm start

The app will run on:
http://localhost:3000
